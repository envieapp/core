## Installation

Install @envie/core with npm

```bash
npm install @envie/core \
  --registry https://npm.pkg.jetbrains.space/bruhabruh/p/envie/npm/
```

Install @envie/core with yarn

```bash
yarn add @envie/core \
  --registry https://npm.pkg.jetbrains.space/bruhabruh/p/envie/npm/
```
